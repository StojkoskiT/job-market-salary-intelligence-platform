# Job Market & Salary Intelligence Platform

## 📌 Overview
This project analyzes global data job market trends, salary distributions, and remote work patterns using real-world data. It provides insights into how experience level, job role, and location impact compensation.

## 🚀 Features
- Salary analysis by job title and experience level
- Remote vs on-site job distribution
- Interactive filtering dashboard (Streamlit)
- Market insights for data professionals

## 📊 Dataset
Data Science Job Salaries dataset (~100k records)

## 🧠 Key Insights
- Senior and executive roles dominate high salary ranges
- Remote work is widely adopted across data roles
- Job titles such as ML Engineer and AI Lead command top salaries
